# IAG-System — 2026

## النظام

منصة المراجعة الداخلية والحوكمة — صحة الفيوم.

## القواعد

- لون الهيدر الموحد: `#0a5c56`
- لا نستخدم Tailwind
- التوثيق والخطط في `docs/`

## هيكل المشروع

| المجلد | المحتوى |
|--------|---------|
| `Frontend/` | ملفات HTML و`assets/` (CSS وJS) |
| `Backend/` | Google Apps Script — يُزامن لاحقاً عبر Clasp |
| `docs/` | التقدم، الخطة، والهندسة المعمارية |

## ملاحظة تشغيلية

العمل اليومي يتم على الملفات المنفصلة داخل `Frontend/` و`Backend/` لتوفير السياق (Tokens) ودقة التعديل. ملف `IAG_Full_Project_Archive.txt` مخصص للأرشفة المرجعية فقط وليس مسار التطوير الاعتيادي.
